## Overview

A JavaFX application for tracking car maintenance records, service reminders, expenses, and failures.

## Key Features

- **Maintenance Records**: Track all service history and repairs
- **Service Reminders**: Get notified about upcoming maintenance needs
- **Expense Tracking**: Monitor maintenance costs over time

## Technologies Used

- **Java**: Core programming language
- **JavaFX**: Desktop user interface framework
