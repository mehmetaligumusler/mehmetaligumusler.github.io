## Overview

A comprehensive management system designed for medical stores, developed as a Java console application. Handles inventory management, sales processing, and reporting features.

## Key Features

- **Inventory Management**: Track medicines, expiry dates, and stock levels
- **Sales Processing**: Process sales transactions and generate receipts
- **Reporting**: Generate sales reports and inventory summaries

## Technologies Used

- **Java**: Core programming language
- **Console Application**: Command-line interface
