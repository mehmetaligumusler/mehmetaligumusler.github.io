## Overview

A national-defense project supported by SSB and TÜBİTAK for detecting manipulated faces. The work combined StyleGAN3-based dataset generation with facial-dynamics analysis.

## Technologies Used

- **Python**: Core development language
- **StyleGAN3**: Generative model for synthetic data
- **OpenCV**: Image processing and analysis
- **Deep Learning**: Neural network architectures for detection
