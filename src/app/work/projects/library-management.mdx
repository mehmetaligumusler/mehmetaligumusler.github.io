## Overview

A comprehensive library management system developed as a console application in C#. Provides complete functionality for managing books, users, and library operations.

## Key Features

- **Book Management**: Add, delete, update, and search for books
- **User Management**: Manage library members and borrowing history
- **Borrowing System**: Track book checkouts and returns

## Technologies Used

- **C#**: Core programming language
- **.NET**: Framework for console applications
