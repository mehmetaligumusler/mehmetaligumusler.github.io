## Overview

A desktop application for managing comic book collections, built with Java, Maven, and JavaFX. Provides an intuitive graphical interface for organizing and tracking comic books.

## Key Features

- **GUI Interface**: User-friendly JavaFX-based interface
- **Collection Management**: Add, update, and search comic books
- **Data Persistence**: Save and load comic book collections

## Technologies Used

- **Java**: Core programming language
- **JavaFX**: GUI framework for desktop applications
- **Maven**: Build automation and dependency management
