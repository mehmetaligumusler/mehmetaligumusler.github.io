## Overview

A real-time virtual clothing try-on system developed with TÜBİTAK 2209-A support. The system uses a camera interface and computer vision to overlay garments on the user.

## Key Features

- **Real-time Processing**: Uses OpenCV and Python for live camera feed processing
- **Virtual Fitting**: Accurately overlays clothing on the user's body
- **User-Friendly Interface**: Simple and intuitive camera-based interface

## Technologies Used

- **Python**: Core programming language
- **OpenCV**: Computer vision library for image processing
