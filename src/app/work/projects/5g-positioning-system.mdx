## Overview

A fingerprint-based location estimation model developed for the TEKNOFEST 2025 5G Positioning Competition. The system predicts location from 5G base-station signal measurements.

## Achievement

**4th Place** - TEKNOFEST 2025 5G Positioning Competition, with a measured **0.20 m MAE**.

## Technologies Used

- **Python**: Data analysis and machine learning
- **Pandas/NumPy**: Data processing
- **Machine Learning**: Predictive modeling
